## 环境搭建

windows 环境, [教程](https://hugo.opendocs.io/installation/windows/)

安装 Hugo 扩展版本

```shell
winget install Hugo.Hugo.Extended
```

安装完成之后重启 shell 或重启 ide, 检查是否安装成功

```shell
hugo version
```

## 安装 dart sass

使用 choco 安装

```shell
choco install sass
```

有些主题需要使用 dart sass 来编译 scss 文件, 所以需要安装 dart sass

检查是否安装成功

```shell
sass --version
```

## 本地创建博客项目

```shell
hugo new site gardenia
```

会在当前文件夹下创建并初始化一个 hugo 项目, 项目名称为 gardenia
进入项目文件夹, 并初始化 git 仓库

```shell
cd gardenia
git init
```

### 添加主题子模块

```shell
git submodule add https://github.com/CaiJimmy/hugo-theme-prism.git themes/prism
```

该命令会在 themes 文件夹下添加 prism 主题子模块

### 本地预览

```shell
hugo server
```

该命令会在本地启动一个服务器, 并监听 1313 端口, 可以在浏览器中访问 <http://localhost:1313> 来预览博客

## github 配置

### 创建 github 仓库, 并推送项目代码

在 github 上创建一个新的仓库, 仓库名称为 gardenia-blog (public, 不初始化 README 文件, 不添加 .gitignore 文件)

将本地项目代码推送到 github 仓库

```shell
git remote add origin https://github.com/yourusername/gardenia-blog.git
git branch -M main
git add .
git commit -m "Initial commit"
git push -u origin main
```

### 配置 github pages

在仓库设置中, 找到 github pages 选项, 选择 main 分支作为 github pages 来源

### 配置自定义域名

![alt text](/images/dns.png)

- 修改阿里云当前 DNS 服务器, 修改成 cloudflare

![alt text](/images/cloudflare.png)

- 阿里云里的 DNS 解析设置全部关闭, 只保留 cloudflare 的 DNS 解析
- cloudflare 里对网站配置路由解析
  ![alt text](/images/cloudflare-route.png)

## 发布 Cloudflare Pages

在 Cloudflare 中创建一个新的 Pages 项目, 项目名称为 gardenia-blog, 选择 github 作为项目来源, 并选择 gardenia-blog 仓库

在 Cloudflare Pages 项目设置中, 找到域名配置选项, 添加自己的域名
